Learn Enumerable Methods With Me

What is Enumerable#cycle?

cycle(n=nil) { |obj|block }

This method calls a block for each element of enum repeatedly n times or forever if none or nil is given.

But what does this mean?

Just like a looping, you can have an array do something again and again within the array. It can continue forever, or you can pass an argument to tell it how many times it should repeat.

You can learn more about cycles by checking out this in Ruby-Docs!
